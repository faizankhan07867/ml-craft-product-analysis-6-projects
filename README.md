# ML Craft E-commerce Analysis – 6 End-to-End Projects  

**Predict Ratings • Predict Prices • Cluster Products • Sentiment Analysis • Visualization**  
Built on real-world handmade craft product data (Pottery, Macrame, Candles, Jewelry)

Live Colab Notebooks → Open & Run in 5 seconds  
GitHub → https://github.com/faizankhan07867/ml-craft-ecommerce-analysis

## 6 Machine Learning Projects

| # | Project                          | Model / Technique                  | Notebook |
|---|----------------------------------|------------------------------------|----------|
| 1 | Product Rating Predictor         | Random Forest Regressor            | [`01-Rating-Predictor.ipynb`](01-Rating-Predictor.ipynb) |

- Python, Pandas, Scikit-learn
- Matplotlib, Seaborn, Plotly
- TextBlob, WordCloud
- Google Colab Ready (no installation needed)

## How to Run
1. Click any `.ipynb` file → "Open in Colab"  
2. Press `Ctrl+Enter` on each cell → sab kuch run ho jayega  
3. Download PDF → submission ready!

## Key Insights Found
- High-rated products → Fast shipping + High seller rating
- Premium cluster → ₹1500+, 4.7+ rating, low volume
- 68% reviews positive → "handmade", "beautiful", "worth it" top keywords
